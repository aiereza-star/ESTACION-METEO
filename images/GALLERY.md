# Gallery of Screenshots

## TFT Physical Display Screens

### 1. Clock Screen
- **Description**: Displays the current time in a large format along with date information. 
- **Data Displayed**: Hour, minute, second, day, month, year.
- **Colors Used**: Dark background with white text for readability.
- **Features Shown**: Easily readable time, possible date format settings.

### 2. Interior Screen
- **Description**: Shows interior temperature, humidity, and other sensor data within the environment.
- **Data Displayed**: Temperature in Celsius, humidity percentage, and CO2 levels.
- **Colors Used**: Blue colors for temperatures, green for humidity items.
- **Features Shown**: Graphical representation of the readings, adjustable thresholds for alerts.

### 3. Exterior Screen
- **Description**: Provides information about the external environment.
- **Data Displayed**: Outdoor temperature, humidity, wind speed.
- **Colors Used**: Sunny yellow for optimal conditions, gray for warnings.
- **Features Shown**: Wind direction indicator, UV index.

### 4. Details Screen
- **Description**: Displays detailed weather statistics for a user-defined period.
- **Data Displayed**: Highs and lows for the day, historical data trends, and forecasts.
- **Colors Used**: Warm oranges for highs, cool blues for lows.
- **Features Shown**: Interactive touch to view different periods, weekly forecasts.

### 5. Forecast Screen
- **Description**: Shows the weather forecast for the next several days.
- **Data Displayed**: Expected temperatures, chance of precipitation, and weather icons.
- **Colors Used**: Bright colors to visually indicate different weather types (rain, sun, clouds).
- **Features Shown**: Swipe to view extended forecasts, notifications for severe weather.

## PWA Web Interface Screens

### 1. Dashboard Screen
- **Description**: Main interface showcasing all critical real-time data.
- **Data Displayed**: Current weather conditions, graphs for temperature trends, alerts.
- **Colors Used**: White background with colorful graphs (green, blue, red).
- **Features Shown**: Customizable widgets, real-time updates, easy navigation.

### 2. Detail Screen
- **Description**: More granular view of specific weather data.
- **Data Displayed**: Detailed statistics including wind patterns, humidity, and temperature variations.
- **Colors Used**: Neutral palette with colorful icons for different data points.
- **Features Shown**: Options to compare real-time data with historical data.

### 3. History Screen
- **Description**: Historical weather data tracking.
- **Data Displayed**: Daily summaries, graphical trends over time.
- **Colors Used**: Soft pastels for graphs, clear labeling.
- **Features Shown**: Ability to filter by date range, export data options.

### 4. Login Screen
- **Description**: User authentication interface.
- **Data Displayed**: Username, password fields.
- **Colors Used**: Simple and clean layout with blue and white contrast.
- **Features Shown**: Option for password recovery and user sign up.

### 5. Config Screen
- **Description**: Settings for user customization.
- **Data Displayed**: User preferences for alerts, units of measurement, themes.
- **Colors Used**: Color-coded sections for easy identification of settings categories.
- **Features Shown**: Live previews of changes, easy save and reset configurations.

---

This gallery provides a comprehensive view of the different interfaces available on the TFT physical display and the PWA web interface, showcasing the capabilities and design of the project.
